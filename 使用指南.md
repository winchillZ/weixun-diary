```bash

# 3. 安装依赖

# 4. 启动开发服务器
npm start

# 5. 浏览器自动打开 http://localhost:3000
```

### 方法二：直接打开构建文件

```bash

# 2. 用任何静态服务器打开（如 nginx、apache）

```

### 方法三：部署到线上

你可以将 `build` 文件夹部署到任何静态托管平台：

- **Vercel**: 拖拽 build 文件夹到 vercel.com

- **GitHub Pages**: 上传 build 文件夹内容
