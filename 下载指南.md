## 🚀 部署到 Vercel（使用下载的文件）

### 步骤：

1. **下载并解压**
   ```bash
   unzip weixun-diary-build.zip
   ```

2. **登录 Vercel**
   ```bash
   vercel login
   ```

3. **部署**
   ```bash
   cd weixun-diary/build
   vercel --prod
   ```

4. **获取网址**
   部署完成后会显示：`https://weixun-diary-xxx.vercel.app`
