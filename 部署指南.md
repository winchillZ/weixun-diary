## 🌐 三种最简单的部署方法

### 方法一：Surge.sh（最快，30秒上线）⭐ 推荐

**Surge** 是一个免费的静态网站托管服务，无需注册，一行命令即可部署！

#### 步骤：

1. **下载构建文件**
   - 将 `/workspace/weixun-diary-build.zip` 下载到你的电脑
   - 解压缩到任意文件夹

2. **安装 Surge**
   ```bash
   npm install -g surge
   ```

3. **部署到线上**
   ```bash
   # 进入解压后的 build 文件夹
   cd build

   # 运行部署命令
   surge
   ```

4. **获取网址**
   - 第一次使用会要求输入邮箱
   - 选择一个域名（如：`weixun-diary.surge.sh`）
   - 部署完成后会显示网址

**优点**：
- ✅ 完全免费
- ✅ 无需注册
- ✅ HTTPS 支持
- ✅ 30秒上线
- ✅ 可自定义域名

---

### 方法二：Netlify Drop（最简单，拖拽即部署）⭐⭐ 强烈推荐

**Netlify** 提供了一个超简单的拖拽部署功能！

#### 步骤：

1. **下载构建文件**
   - 将 `/workspace/weixun-diary-build.zip` 下载到你的电脑
   - 解压缩得到 `build` 文件夹

2. **拖拽部署**
   - 访问：https://app.netlify.com/drop
   - 或者访问：https://www.netlify.com/ 并点击 "Drop folder"
   - 将 `build` 文件夹直接拖拽到网页上
   - 等待几秒钟，完成！

3. **获取网址**
   - Netlify 会自动生成一个随机网址（如：`https://random-name.netlify.app`）
   - 你可以在设置中修改域名

**优点**：
- ✅ 完全免费
- ✅ 无需命令行
- ✅ 自动 HTTPS
- ✅ CDN 加速
- ✅ 可绑定自定义域名
- ✅ 每次更新只需重新拖拽

---

### 方法三：Vercel（最专业，适合长期使用）

**Vercel** 是 React 官方推荐的部署平台，性能最佳！

#### 步骤：

1. **注册 Vercel 账号**
   - 访问：https://vercel.com
   - 使用 GitHub/GitLab/Bitbucket 账号登录

2. **部署方式一：拖拽部署**
   - 登录后，点击 "Add New..." → "Project"
   - 选择 "Import Git Repository" 或直接拖拽文件夹
   - 将 `build` 文件夹拖拽上传

3. **部署方式二：命令行（推荐）**
   ```bash
   # 安装 Vercel CLI
   npm install -g vercel

   # 进入 build 文件夹
   cd build

   # 部署
   vercel

   # 按照提示操作，一路回车即可
   ```

4. **获取网址**
   - Vercel 会生成一个网址（如：`https://weixun-diary.vercel.app`）
   - 你可以在设置中修改项目名和域名

**优点**：
- ✅ 免费额度充足
- ✅ 全球 CDN 加速
- ✅ 自动 HTTPS
- ✅ 性能最佳
- ✅ 支持自定义域名
- ✅ 每次更新只需运行 `vercel` 命令

## 📱 其他部署选项

### GitHub Pages（免费，适合开源项目）

1. 在 GitHub 创建仓库
2. 上传 `build` 文件夹内容
3. 在仓库设置中启用 GitHub Pages
4. 访问 `https://your-username.github.io/repo-name`

### Cloudflare Pages（免费，全球加速）

1. 访问：https://pages.cloudflare.com
2. 连接 Git 仓库或直接上传
3. 自动部署和加速

### 阿里云 OSS / 腾讯云 COS（国内访问快）

1. 创建对象存储桶
2. 上传 `build` 文件夹内容
3. 配置静态网站托管
4. 绑定自定义域名
